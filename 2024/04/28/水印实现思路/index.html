水印实现思路

本文最后更新于：2024年4月28日 晚上

使用

props 可传入文字 content 或者图片 image，做为水印内容
props 可传入 container 做为水印容器

实现

  1. 像 container 中插入水印元素
    获取 WaterMark 元素的 container ，创建一个 div 做为水印元素，将 container 的样式进行改写，position 设置为 relative ，把水印 div append 到 container ，设置水印 div 样式 position 为 absolute，大小设置与 container 相同。

  2. 通过 canvas 绘制水印
    使用 canvas 使用 fillText（水印内容为文字）或者 drawerImage（水印内容为图片）绘制水印。然后通过 canvas.toDaraURL 将 canvas 转化为图片 base64 地址。获取单个水印内容图片

  3. 为水印元素添加水印背景
    获取到 base64 图片 url 后，水印元素设置 backgroud-position 为 0 0 ，设置 background-repeat 为 repeat ，设置 background-size 为水印内容图片的 width、height ，设置 background-image 为水印图片 base64 url 。

  4. 通过 mutationObserver 去监听 container 的变化，当水印元素本身的 attributes 发生变化时重新绘制水印；当水印元素被删除时，也要重新绘制水印

Tips

  1. canvas 绘制水印的过程中，需要注意 canvas 的宽高，水印内容图片的宽高，水印元素的宽高，通过 ctx.measureText(content) 可以拿到实际渲染的大小。旋转后的宽高需要计算
